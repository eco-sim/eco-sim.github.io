Your native solution to read text from your photos in bulk

Unleash Trapped Text

TextWizardPro allows you to loop through images natively on your computer and uses OCR technology to convert words found in images into a text file for you to use. It works with almost any image file type, including PDFs. It can save hours of hand by hand translation, without the hassle of uploading photos. Install now.

Install Tips/Application Guide

TextWizardPro is new. This will cause many systems to not recognize it as a publisher, flagging it as a virus. TextWizardPro is not a virus. Feel free to bypass any attempts to stop the installation process.

When running the application, you will be asked for a product key. Product Keys are purchased from TextWizardPro vendors, contact carson@couckama.net for more information. Product keys are only needed at the installation stage.

Photos ready for conversion need to be placed in the TextWizardPictures folder, after running the application the ready text files will be placed in the TextWizardTexts folder. Both of these folders can be found in your WizardToolkit, which you can place wherever you would like during installation.
